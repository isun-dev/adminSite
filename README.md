# adminSite
- Ant Design Pro + Spring Boot 를 활용해서 관리자 사이트 프로젝트 진행.
- 백엔드 위주 개발
- 프론트 엔드는 Ant Design Pro UI 템플릿 사용, 단, Typescript 로 커스터마이징도 해볼 예정

# 기술 스택
- Typescript => Ant Design Pro
- Spring Boot, Java
- DataBase => 고려중
- AWS
- 자동 배포 => GitAction, Jenkins
- 그 외 기술 스택은 고려중
